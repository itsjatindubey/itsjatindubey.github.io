# Jatin Dubey Portfolio

A GitHub Pages-ready personal portfolio inspired by a clean product-management landing page.

## How to publish on GitHub Pages

1. Create a GitHub repository named `jatindubey71.github.io`.
2. Upload `index.html` and `style.css` to the repository root.
3. Go to **Settings → Pages**.
4. Select branch: `main`, folder: `/root`.
5. Your site will be live at `https://jatindubey71.github.io/`.

## What to edit

- Replace the profile initials `JD` with your image if needed.
- Update project links.
- Add your real resume link.
- Confirm LinkedIn and GitHub URLs.
